# RPG Site Builder - Kuar-Tor Style

Este é um construtor de sites para RPG de mesa, inspirado na estética de **Kuar-Tor**. Ele permite que você edite textos, imagens e cores em tempo real diretamente no navegador.

## 🚀 Como usar

1. **Abra o `index.html`** no seu navegador.
2. Clique no **botão circular (ícone de lápis)** no canto inferior direito para abrir o painel de edição.
3. **Edite o Texto:** Clique em qualquer bloco de texto e comece a digitar.
4. **Edite Imagens/Links:** Clique no botão "Mudar Imagem" ou "Editar Link" que aparece ao passar o mouse sobre os elementos.
5. **Personalize:** Use o seletor de cores para mudar o tema (cor de destaque).
6. **Salve:** Clique em "Salvar Rascunho" para manter as alterações localmente no seu navegador.
7. **Exporte:** Quando estiver satisfeito, clique em "Exportar HTML" para baixar uma versão limpa e otimizada (sem o editor) para publicar.

## 🌐 Publicando no GitHub Pages

Para fazer o site rodar no seu GitHub:

1. Suba os arquivos (`index.html`, `style.css`, `main.js`) para o seu repositório.
2. Vá em **Settings > Pages**.
3. Selecione a branch `main` e a pasta `/ (root)`.
4. Clique em **Save**.

Após alguns minutos, seu site estará online!

## 🛠 Tecnologias
- HTML5 / Vanilla JavaScript
- Tailwind CSS (via CDN)
- Google Fonts (Cinzel & Roboto Slab)
- LocalStorage para persistência local
